## üöÄ Quick Start Guide

### 1. Environment Setup
```bash
# Install dependencies
cd loa-core
pip install -r requirements.txt

# Setup environment
cp .env.example .env
# Edit .env with your API keys
```

### 2. Start Loa Brain API
```bash
cd loa-core
python loa_api.py
```
API runs on: http://localhost:8000

### 3. Configure N8n Integration
Add `mcp_config.json` to Windsurf MCP configuration:
```json
{
  "mcpServers": {
    "n8n-mcp": {
      "command": "npx",
      "args": ["-y", "supergateway", "--streamableHttp", "https://ixlmnts.app.n8n.cloud/mcp-server/http", "--header", "authorization:Bearer YOUR_TOKEN"]
    }
  }
}
```

### 4. Start Sales Automation
```bash
cd loa-core
python sales_automation.py
```

### 5. Update All Platforms
```bash
cd loa-core
python update_platforms.py
```
